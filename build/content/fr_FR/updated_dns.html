Blokada Slim v4.3

Votre version de Blokada vient d'être mise à jour !

Il existe deux éditions de Blokada. L'application que vous utilisez actuellement, disponible sur Google Play, est appelée Blokada Slim (ancien nom: DNS Changer by Blokada). Ce n'est pas un bloqueur de publicité. Il vous donne la possibilité de changer vos serveurs DNS, et aussi d'utiliser notre tout nouveau service VPN (facultatif). La version open source de Blokada est disponible sur notre site web.

Vous avez peut-être remarqué la nouvelle interface utilisateur dans cette mise à jour. C'est la nouvelle interface v4, et nous espérons que vous l'aimerez ! Pour plus d'informations sur les modifications, consultez le fichier changelog.
