Blokada Slim v4.5

Je Blokada is zojuist bijgewerkt!

Er zijn twee edities van Blokada. Degene die je gebruikt, beschikbaar op Google Play, heet Blokada Slim (voorheen: DNS changer door Blokada). Dit is geen adblocker. Het geeft je de mogelijkheid om je DNS-servers te wijzigen en ook om onze gloednieuwe VPN-service (optioneel) te gebruiken. De open source versie van Blokada is beschikbaar op onze website.

The 4.5 release is a maintenance update with improvements for stability, and user experience. For more info about the changes check our changelog.
