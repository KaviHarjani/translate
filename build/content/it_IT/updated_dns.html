Blokada Slim v4.5

La tua Blokada è stata appena aggiornata!

Ci sono due versioni di Blokada. Quella che stai usando, disponibile su Google Play, è chiamata Blokada Slim (vecchio nome: DNS changer by Blokada). Questa non è un adblocker. Ti dà la possibilità di cambiare i tuoi server DNS, inoltre anche di utilizzare il nostro nuovo servizio VPN (opzionale). La versione open source di Blokada è disponibile sul nostro sito.

The 4.5 release is a maintenance update with improvements for stability, and user experience. For more info about the changes check our changelog.
