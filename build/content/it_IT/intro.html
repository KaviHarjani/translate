Benvenuti e grazie per aver utilizzato Blokada!

Blokada è gratuita, open source e rispetta la tua privacy. Utilizzandola, accetti i Termini di Servizio e l'Informativa sulla Privacy, che puoi sempre trovare qui.

Saremmo lieti di aiutarti con i primi passi:

  1. Per impostazione predefinita, riceverai una notifica quando una pubblicità viene bloccata. È possibile modificare questa impostazione disabilitando l'opzione Notifiche.
  2. Blokada è impostata per l'avvio all'accensione del telefono. You can turn if off by tapping on Start on boot.
  3. Blokada will be running only when you are connected to the internet. You can change it by setting Online only to off.
  4. In the Blacklist you can find the lists that contain the ads-servers to block.
  5. In the Whitelist you can define which apps or websites shouldn't be checked and blocked.
  6. Tapping in the top right corner you can find more options and the Help menu where you can get detailed information about the different options in Blokada.
