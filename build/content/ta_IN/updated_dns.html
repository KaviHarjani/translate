Blokada Slim v4.6

உங்கள் Blokada புதுப்பிக்கப்பட்டது!

Blokada-வில் இரண்டு பதிப்புகள் உள்ளது. நீங்கள் பயன்படுத்துவதன் பெயர் Blokada Slim. இது விளம்பர முடக்குதலில் பயன்படாது. உங்கள் DNS Servers-ஐ மாற்றி எங்கள் VPN Service-யும் பயன்படுத்தலாம். Blokada-வின் கட்டற்ற மென்பொருள் பதிவு எங்கள் வலைதளத்தில் இருந்து பெற்றுக்கொள்ளலாம்.

The 4.6 release is a maintenance update with improvements for stability, and user experience. இதைப் பற்றி மேலும் அறிய இங்கு பார்க்கவும்.
