Willkommen und vielen Dank, dass du Blokada DNS benutzt!

Blokada ist kostenlos, open-source und respektiert deine Privatsphäre. Wenn du es benutzt, akzeptierst du unsere Nutzungsbedingungen und Datenschutzbestimmungen, welche du immer hier finden kannst.

Wir begleiten dich gerne bei deinen ersten Schritten:

  1. Der Standard-DNS-Server wird derzeit benutzt. Du kannst dies ändern, indem du auf das Standard-DNS-Symbol tippst.
  2. Blokada wird beim Start Ihres Telefons gestartet. You can turn if off by tapping on Start on boot.
  3. Blokada will be running only when you are connected to the internet. You can change it by setting Online only to off.
  4. In the Whitelist you can define which apps shouldn't use your DNS servers. They will use default device DNS servers instead.
  5. Tapping in the top right corner you can find more options and the Help menu where you can get detailed information about the different options in Blokada.
