Pemblokiran iklan

Fitur Blokada yang paling dikenal adalah kemampuannya untuk memblokir iklan. Blokada adalah lapisan tipis antara antarmuka jaringan (wifi atau mobile telepon) dan aplikasi yang Anda gunakan. Agar dapat menyaring setiap trafik keluar, Blokada akan membuat koneksi VPN lokal (tanpa layanan pihak ketiga) dan hanya memblokir setiap permintaan ke host yang ditentukan dalam filter yang ada di dalam blacklist. Pemblokiran iklan ini dilakukan dengan cara menghasilkan laporan sederhana berisi "Nama host tidak ditemukan" jika suatu aplikasi mencoba mengakses sebuah host yang telah tercantum di dalam blacklist. Dengan cara ini, Blokada menghemat paket data, baterai, dan waktu Anda: tidak ada pengalihan data yang terjadi. Filter diperbarui secara otomatis. Anda dapat mengkonfigurasi Blokada dengan filter dan whitelist buatan Anda sendiri untuk memastikan layanan dan/atau aplikasi yang Anda ingin gunakan tidak diblokir dan Anda tidak mengalami gangguan dalam menggunakan perangkat Anda.
Fitur ini tersedia secara gratis, dan akan tetap gratis untuk selamanya. Selamat menggunakan Blokada!

Jika suatu aplikasi tidak bekerja sebagaimana mestinya, tidak memunculkan kontennya atau Anda mengalami perilaku tidak lazim lainnya, silakan buka menu Blokada dengan cara menggeser ke atas, tekan Pemblokiran Iklan kemudian pilih Host yang Diizinkan, tekan tanda "+" di bagian atas dan tambahkan nama host atau file berisi daftar semua host yang Anda ingin izinkan untuk diakses.

Anda dapat memblokir situs yang tidak ada dalam daftar host satu per satu secara manual. Untuk melakukannya, buka menu Blokada dengan cara menggeser ke atas, tekan Pemblokiran Iklan, pilih Host yang Diblokir kemudian tekan tanda "+" di bagian atas layar. Di sini Anda dapat menambahkan satu nama host atau file berisi daftar semua host yang ingin Anda blokir.

VPN

Di Blokada v4 kami memperkenalkan layanan baru, yaitu VPN yang dikelola oleh kami sendiri. Layanan ini kami perkenalkan untuk memastikan keamanan dan anonimitas Anda ketika berselancar di internet, di manapun Anda berada, apapun yang Anda lakukan. Silakan lihat artikel di blog kami untuk mempelajari lebih jauh mengenai VPN.
Tarif untuk layanan VPN kami adalah 5 euro/bulan. Silakan baca Ketentuan Layanan kami di sini dan Kebijakan Privasi di sini terlbih dahulu sebelum Anda memutuskan untuk menggunakan layanan VPN kami. Di bagian ini Anda akan bisa mengelola akun VPN Anda. Dari sini Anda akan bisa:

  • Mengaktifkan akun
  • Memulihkan akun Anda
  • Memilih gateway yang akan digunakan

Sebagai mekanisme pembayaran, kami menggunakan Stripe dan informasi yang tersimpan pada saat Anda melakukan pembayaran hanya akan digunakan untuk memverifikasi pembayaran dan ketika Anda mengaktifkan akun Anda, sehingga Anda bisa mengelola akun Anda secara aman.

Mengganti DNS

Fitur lain yang sering digunakan di Blokada adalah DNS Changer. Blokada menyediakan daftar server DNS yang bisa Anda pilih
DNS adalah singkatan dari Domain Name Server dan server inilah yang bertanggung jawab untuk menyambungkan nama domain (misalnya: contoh.com) ke alamat IP, dan digunakan sebagai alat berkomunikasi antar perangkat. Penyedia layanan internet Anda memiliki server DNS sendiri; Blokada dikonfigurasi untuk menggunakan server tersebut. Server yang ada di dalam daftar ini bisa digunakan jika Anda menduga ISP Anda mengumpulkan catatan tentang penggunaan internet Anda, atau Anda takut akan risiko terhadap privasi Anda atau jika beberapa situs web atau layanan yang Anda akses diblokir. Tidak ada server DNS yang bisa disebut "terbaik", semuanya bergantung pada preferensi dan lokasi geografis Anda, dengan demikian jika Anda ingin menggunakan server DNS pilihan Anda sendiri, harap baca deskripsinya terlebih dahulu, cari tahu semua hal terkait server DNS tersebut dan baca semua ulasan penggunanya.
Fitur DNS Changer ini tetap bisa berfungsi dengan atau tanpa VPN, maka dari itu apapun DNS yang Anda pilih, server itulah yang akan dipakai. Harap diingat juga bahwa Blokada v4 masih belum mendukung DoH/DoT, fitur ini akan kami implementasikan segera setelah v4 dirilis.

Apa saja pengaturan yang tersedia di Blokada?

Seperti yang Anda lihat, Blokada memiliki beberapa pilihan di halaman Pengaturan. Berikut penjelasannya:

  • Notifikasi: jika diaktifkan, Anda akan diberi tahu saat iklan diblokir. Notifikasi ini diaktifkan secara otomatis, tetapi bisa dinonaktifkan jika Anda merasa notifikasi yang muncul terlalu banyak.
  • Mulai saat boot: ketika diaktifkan, Blokada akan mulai bekerja setelah sistem melakukan boot. Harap diingat ada konfigurasi di tingkat sistem untuk pilihan ini, oleh karena itu kami tidak dapat menjamin Blokada bisa langsung aktif saat boot. Jika hal ini terjadi, periksa pengaturan ponsel Anda. Konfigurasi standar diaktifkan, jika Anda ingin memulai Blokada secara manual, nonaktifkan pilihan ini.
  • Lokasi Penyimpanan Pengaturan: di mana Anda ingin menyimpan konfigurasi yang telah Anda buat: penyimpanan internal atau eksternal? Jika Anda memilih penyimpanan eksternal, file konfigurasi Anda akan tersimpan di direktori /Download/Blokada.
  • Animasi latar belakang: animasi gelombang di latar belakang ini bisa Anda aktifkan atau nonaktifkan. Gelombang berwarna putih mewakili nama-nama host yang diizinkan, sementara gelombang berwarna oranye adalah host yang diblokir.
  • Permintaan Log: jika diaktifkan, Blokada akan membuat file berformat CSV berisi log permintaan akses yang telah diblokir dan/atau diizinkan untuk diakses oleh Blokada.
  • Keep Alive: pilihan ini berfungsi untuk memastikan Blokada tidak berhenti atau dihentikan oleh sistem. Harap diingat sistem Anda dapat mengambil alih pengaturan ini, pastikan pengaturan Anda akan mengizinkan Blokada untuk aktif di latar belakang.
  • Watchdog: jika diaktifkan maka Blokada akan berhenti bekerja ketika Anda tidak terkoneksi ke internet, dan akan mulai bekerja kembali ketika Anda sudah terkoneksi. Jika dinonaktifkan, Blokada akan selalu berjalan di latar belakang (meskipun tidak ada jaringan). Harap diingat bahwa pilihan ini mungkin tidak berfungsi pada perangkat tertentu.
  • Penggugah Agresif: aktifkan pilihan ini jika perangkat Anda menghentikan atau mematikan Blokada dikarenakan fitur penghemat daya. Walaupun pilihan ini ditujukan untuk tetap mengaktifkan Blokada di sebagian besar perangkat, sistem perangkat Anda akan selalu memiliki prioritas teratas dan bisa menonaktifkan Blokada jika tidak dikecualikan dari fitur optimisasi memori atau baterai.

Saya tidak bisa mengaktifkan Blokada setelah saya menginstalnya.

Ada dua penyebab umum untuk situasi seperti ini. Pertama, periksa apakah Anda memiliki koneksi VPN yang aktif dan apakah pilihan VPN selalu-hidup diaktifkan. Untuk melakukannya, buka pengaturan smartphone Anda, cari VPN di bagian jaringan, tekan ikon roda gigi di sebelah nama VPN pilihan Anda. Nonaktifkan, kemudian lepaskan dari sambungan VPN aktif dan cobalah untuk me-restart Blokada.

Jika Anda tidak memiliki koneksi VPN lain yang sedang aktif, periksa apakah Watchdog diaktifkan di Pengaturan Lanjut, jika pengaturan tersebut aktif, periksa kembali apakah Anda tersambung ke jaringan seluler/Wi-Fi. Blokada tidak bekerja ketika Watchdog sudah diaktifkan dan Anda tidak terkoneksi ke internet.

Jika langkah-langkah tersebut tidak membantu, start ulang telepon Anda, kadangkala hal tersebut juga bisa membantu menyelesaikan masalah.

Blokada memblokir lebih sedikit iklan dibandingkan dengan sebelumnya.

Jika Anda menduga Blokada memblokir lebih sedikit iklan daripada sebelumnya, harap segarkan daftar Anda. Untuk melakukannya, tekan Blacklist, lalu di sudut kanan atas pada ikon equalizer, pilih Segarkan filter dan tekan pada Lanjutkan. Biasanya hal itu terjadi karena server baru ditambahkan ke daftar server yang sudah ada dan tidak (atau belum) diidentifikasi dan ditambahkan ke dalam blacklist, atau daftar baru tidak diunduh ke ponsel Anda, sehingga iklan tersebut belum bisa diblokir.
Sebagai alternatif, Anda dapat mengaktifkan lebih banyak daftar host atau menambahkan filter yang lebih ketat di menu Blacklist.

Harap diingat bahwa Blokada menggunakan daftar pihak ke-3, oleh karena itu jika sebelum pembaruan Blokada berfungsi sebagaimana mestinya dan masih memblokir iklan, maka Blokada seharusnya tetap berfungsi.

Blokada tidak memblokir iklan di Chrome.

Jika Anda menggunakan Google Chrome, Anda perlu menonaktifkan Data Saver dan Async DNS. Google menggunakan dua teknologi tersebut secara default dan dua hal itu mengganggu kinerja Blokada dalam hal pemblokiran iklan.
1. Untuk menonaktifkan Data Saver, buka Pengaturan, cari "Data Saver" atau "Lite Mode", kemudian nonaktifkan.
2. Untuk menonaktifkan Async DNS, ketik "chrome://flags" (tanpa tanda kutip) di Google Chrome, kemudian cari dan nonaktifkan fitur Asynchronous DNS.
3. Buka pengaturan perangkat Anda, kemudian Aplikasi -> Chrome -> Penyimpanan lalu bersihkan cache.
4. Reboot!

Iklan di YouTube tidak diblokir!

Memblokir iklan di YouTube sangat sulit karena Google menggunakan enkripsi dan host yang dihasilkan secara acak untuk menyajikan iklan. Oleh karena itu, lebih mudah untuk memblokir iklan dalam aplikasi YouTube dibanding dengan menggunakan piranti lunak eksternal (misalnya Blokada). NewPipe dan YouTube Vanced adalah alternatif aplikasi YouTube resmi yang luar biasa dan memiliki banyak fitur, silakan coba mana yang lebih cocok untuk Anda.

Blokada berhenti berfungsi/bekerja setelah beberapa saat.

Sebagai langkah pertama, aktifkan Keep Alive di Blokada. Dalam kasus ini, Blokada akan mencoba untuk tetap online, tetapi harap diingat, sistem Anda masih dapat mempengaruhi perilaku ini. Jika Anda telah mengatur Blokada untuk selalu aktif, dan Anda masih melihat bahwa Blokada berhenti setelah beberapa waktu, periksa pengaturan sistem Anda dan matikan setiap pengoptimalan baterai dan memori untuk Blokada. Sayangnya pengaturan ini berbeda pada setiap sistem, oleh karena itu kami tidak bisa menunjukkan dengan pasti di mana untuk melakukannya. Silakan periksa pengaturan telepon Anda. Masalah tersebut bisa juga dikarenakan oleh aplikasi pembersih memori yang Anda gunakan. Harap masukkan Blokada ke dalam whitelist atau daftar pengecualian pada aplikasi pembersih memori yang Anda gunakan. Jika langkah-langkah yang sudah disebutkan tadi tetap tidak membantu dalam kasus Anda, laporkan sebagai bug (lihat di bawah untuk cara melakukannya).

Saya tidak dapat mengimpor daftar EasyList/AdBlock Plus ke dalam Blokada.

Jika Anda mencoba untuk mengimpor daftar (misalnya EasyList) yang bekerja sangat baik dengan ekstensi AdBlock Plus di PC atau notebook Anda, Blokada akan melaporkan pesan kesalahan. Hal ini dikarenakan Blokada memakai metode lain untuk memblokir iklan. ABP memblokir dan menyembunyikan elemen atau beberapa bagian dari sebuah situs, namun Blokada tidak bisa melakukan hal tersebut karena Blokada bukanlah plug-in dalam browser. Blokada bekerja pada lapisan yang lebih rendah dan memblokir akses host.

Hal-hal yang tidak dapat dikonfigurasi:

  • Terkini: jika Anda melihat ini, berarti Anda sedang menggunakan Blokada versi terbaru. Terus perbarui Blokada karena versi terbaru memiliki lebih sedikit bug dan lebih kaya fitur!
  • # iklan diblokir: jumlah konten yang tidak diinginkan yang sudah diblokir oleh Blokada sejak pertama kali diaktifkan
  • # host: jumlah server yang menyajikan iklan, malware, dan konten yang tidak diinginkan lainnya

Apa yang bisa saya lakukan jika masih ada iklan yang muncul di perangkat saya?

Beberapa iklan akan tetap bisa lolos, tidak peduli apapun pemblokir iklan yang Anda gunakan. Khususnya iklan YouTube atau beberapa iklan popup layar penuh yang sulit untuk diblokir. Jika menurut Anda iklan lebih sering muncul dibandingkan sebelumnya, coba ikuti langkah-langkah ini:

  • Periksa apakah ikon VPN berbentuk kunci ada di status bar di bagian atas. Jika tidak, aktifkan Blokada.
  • Buka Blokada dan periksa apakah sudah aktif dan tidak melaporkan adanya kesalahan.
  • Pastikan penghitung host tidak sama dengan nol. Jika angka host yang ditampilkan adalah 0, pilih beberapa filter blacklist sehingga iklan dapat diblokir. Jika angka yang tertera masih juga 0 (nol), berikan izin kepada Blokada untuk mengakses media penyimpanan di bagian Pengaturan Sistem - Aplikasi - Blokada, kemudian hapus data dan cache kemudian reboot perangkat Anda.
  • Pastikan juga Anda tidak memasukkan terlalu banyak aplikasi ke dalam whitelist - hal ini bisa jadi penyebab kenapa masih ada iklan di perangkat Anda.
  • Coba mengatur ulang filter Anda: buka 'Ad blocking', kemudian pilih 'Pengaturan Daftar', pilih 'Daftar filter' (bukan Sinkronkan), pilih BUANG.
  • Jika Anda melihat iklan di browser Anda, nonaktifkan segala jenis kompresi data atau pengoptimalan di pengaturan browser.
  • Nonaktifkan Blokada, tutup (dengan menggesernya keluar dari layar Baru), dan aktifkan kembali.
  • Jika Anda belakangan ini mengubah setelan (filter, whitelist, dll.), harap diingat bahwa perlu waktu sebelum konfigurasi yang baru mulai aktif dan bekerja. Tunggu sebentar.
  • Hapus data aplikasi untuk Blokada (dalam pengaturan sistem), atau instal ulang.
  • Mulai ulang/reboot perangkat Anda.
  • Jika langkah-langkah di atas tetap tidak membantu, laporkan bug (lihat di bawah).

Saya tidak bisa menginstal pembaruan, apa yang bisa saya lakukan?

Pertama-tama, silakan periksa apakah versi Android yang Anda miliki adalah versi 5 atau lebih baru. Android versi 4 dan versi yang lebih lama tidak didukung oleh Blokada 3.0 dan versi selanjutnya. Untuk melihat versi Android perangkat Anda, buka Pengaturan ponsel Anda, geser ke Tentang ponsel dan cari versi Android

Jika perangkat Anda menggunakan Android versi 5 atau lebih baru, pastikan Anda menginstal pembaruan dari sumber yang sama ketika Anda pertama kali menginstal Blokada. Setiap repositori (seperti FDroid) menambahkan kunci enkripsi mereka masing-masing, oleh karenanya pembaruan dari sumber yang berbeda memiliki kemungkinan untuk gagal.

Mulai ulang/reboot perangkat Anda. Mungkin ini terdengar aneh, tetapi terkadang hal tersebut bisa membantu. Jika langkah tersebut masih tidak menyelesaikan masalah, hapus Blokada dan pasang kembali.

Jika langkah-langkah tadi tetap tidak berhasil, silakan bergabung dengan grup Telegram kami di go.blokada.org/chat untuk mendapatkan bantuan lebih lanjut, atau melaporkan bug (lihat di bawah).

Dapatkah saya menyembunyikan notifikasi VPN?

Sayangnya tidak. Beberapa perangkat mungkin menampilkan notifikasi VPN yang sedikit mengganggu (ikon kunci) ketika VPN aktif, dan tidak mungkin untuk menyembunyikannya. Jika memungkinkan, cobalah memperbarui perangkat Anda ke Android 8 (Oreo), karena versi Android yang lebih baru tidak memiliki laporan status ini.

Bagaimana cara melaporkan masalah, ide atau saran?

Jika pertanyaan Anda tidak bisa terjawab di sini, silakan bergabung dengan grup Telegram kami di tautan go.blokada.org/chat, komunitas kami siap membantu Anda. Atau cukup pilih Umpan Balik dari menu di sudut kanan atas dan ikuti petunjuknya. Harap sediakan sebanyak mungkin informasi dan sebutkan langkah-langkah yang telah Anda lakukan untuk mereproduksi masalah yang sama - hal itu akan sangat membantu kami dalam memecahkan masalah Anda! Jika Anda adalah pengguna mahir, Anda bisa juga langsung membuka pelacak masalah di GitHub.

Bisakah Blokada melihat data pribadi saya?

Tidak. Blokada tidak membaca atau mengutak-atik informasi pribadi Anda, hanya memonitor host mana yang mengajukan permintaan koneksi di perangkat Anda dan memblokir host yang terdaftar di filter. Setiap kali Blokada melakukan pemblokiran, Anda akan mendapat pemberitahuan (yang bisa Anda matikan kapan saja). Blokada tidak dapat mengakses lalu lintas data terenkripsi Anda.

Apakah Blokada boros baterai?

Tidak. Blokada hanya menggunakan sedikit daya, namun hal ini tergantung pada pengaturan Anda, dan pada sebagian besar perangkat baru, terutama pada Marshmallow (Android 6) atau yang lebih baru, Anda tidak akan mengalami penurunan baterai yang signifikan. Namun, beberapa perangkat yang tidak beruntung mungkin mengalami pengurangan daya baterai yang cukup signifikan.

Akankah Blokada memperlambat kinerja perangkat saya?

Tidak. Blokada bisa sangat efisien, tergantung pada pengaturan Anda (jumlah host di Daftar Hitam dll). Bahkan, kinerja perangkat dan masa pakai baterai Anda mungkin terlihat meningkat, karena Blokada menghindarkan perangkat Anda dari mengunduh dan memproses konten yang tidak Anda butuhkan atau tidak pedulikan. Namun, pengguna yang masih memakai beberapa perangkat yang lebih tua mungkin mengalami penurunan performa.

Apakah Blokada benar-benar gratis untuk digunakan?

Ya! Bkokada bermula sebagai proyek hobi, dan merupakan salah satu cara saya memberi sumbangsih kepada komunitas. Blokada bersifat open source, dengan fitur penyaring/pemblokir iklan yang gratis, dan akan tetap gratis selamanya. Namun, karena pengembangan Blokada membutuhkan banyak waktu, kami harap Anda mempertimbangkan mendukung proyek ini dengan cara apa pun yang Anda bisa (lihat di bawah).

Blokada bagus, bagaimana saya bisa membantu proyek ini?

Ada banyak cara untuk mendukung Blokada, baik secara finansial ataupun cara-cara lainnya. Bantuan dalam bentuk apapun sangat penting untuk memastikan proyek ini dapat tetap berjalan. Silakan buka halaman ini untuk informasi lebih lanjut, atau bergabung dengan obrolan kami.

Saya punya pertanyaan lain, kemana saya bisa bertanya?

Jika Anda tidak menemukan solusi untuk masalah Anda di sini, silakan bergabung dengan kanal Telegram kami. Komunitas Blokada siap membantu Anda. Jika Anda tidak berkenan menggunakan Telegram, silakan gunakan opsi Umpan Balik dari menu atas, atau jika Anda memiliki pertanyaan yang lebih teknis, Anda bisa langsung menuju ke pelacak masalah di GitHub.
