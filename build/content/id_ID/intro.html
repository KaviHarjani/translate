Selamat datang dan terima kasih telah menggunakan Blokada!

Blokada gratis dan open source dan menghormati privasi Anda. Dengan menggunakannya, Anda menyetujui Ketentuan Layanan dan Kebijakan Privasi, yang selalu dapat Anda temukan di sini.

Kami ingin membantu Anda dengan langkah pertama Anda:

  1. Secara standar, Anda akan diberi tahu saat iklan diblokir. Anda dapat mengubahnya dengan menyetel Notifikasi ke posisi mati.
  2. Blokada diatur untuk memulai saat telepon Anda dinyalakan. You can turn if off by tapping on Start on boot.
  3. Blokada will be running only when you are connected to the internet. You can change it by setting Online only to off.
  4. In the Blacklist you can find the lists that contain the ads-servers to block.
  5. In the Whitelist you can define which apps or websites shouldn't be checked and blocked.
  6. Tapping in the top right corner you can find more options and the Help menu where you can get detailed information about the different options in Blokada.
