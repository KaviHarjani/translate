Selamat datang dan terima kasih telah menggunakan Blokada DNS!

Blokada gratis dan open source. Blokada juga menjunjung tinggi privasi Anda. Dengan menggunakan Blokada, Anda secara otomatis setuju dengan Persyaratan Layanan dan Kebijakan Privasi kami, yang bisa Anda pelajari di sini.

Ijinkan kami membantu Anda melakukan pengaturan awal:

  1. Server DNS yang digunakan saat ini adalah server DNS bawaan. Anda bisa mengubahnya dengan mengetuk ikon DNS Bawaan.
  2. Blokada diatur sedemikian rupa untuk mulai bekerja ketika Anda menyalakan smartphone Anda. Anda bisa mematikannya dengan mengetuk Mulai saat boot.
  3. Blokada hanya akan bekerja ketika smartphone Anda terkoneksi dengan internet. Anda bisa mengubahnya dengan mematikan Hanya Online.
  4. Di dalam Whitelist Anda bisa menentukan aplikasi apa saja yang tidak bisa menggunakan server DNS pilihan Anda. Aplikasi yang dimasukkan ke dalam Whitelist akan menggunakan server DNS yang ditentukan perangkat Anda.
  5. Ketuk bagian pojok kanan atas untuk melihat lebih banyak opsi pengaturan dan menu Bantuan di mana Anda bisa menemukan informasi lebih lanjut mengenai opsi apa saja yang tersedia di Blokada.
