Apakah Partner Gateways itu?

Partner Gateways adalah server yang disediakan oleh mitra kami. Dengan demikian, server tersebut tidak dikelola sepenuhnya oleh Blokada, sehingga ada kemungkinan server tersebut tidak menyediakan tingkat kualitas yang sama seperti gateway resmi milik kami. Mitra kami adalah orang-orang baik yang menyediakan server milik mereka untuk mendapatkan keuntungan. Sebagai panduan, Anda disarankan untuk tidak menggunakan gateway milik mitra kami untuk sebagian besar aktifitas pribadi Anda. Ada beberapa keuntungan penting ketika menggunakan partner gateways:

  • Server tersebut di-hosting di jaringan konsumen biasa, dan sebagai akibatnya, tidak bisa diblokir oleh penyedia layanan (misalnya perusahaan penyedia layanan streaming).
  • Partner Gateways memungkinkan Anda mengakses lokasi tambahan di luar yang sudah disediakan oleh gateway resmi.
  • Lalu lintas data online Anda akan tetap terenkripsi.

Harap diingat bahwa fitur ini masih dalam tahap percobaan, dan bisa saja tidak berfungsi sesuai dengan keinginan Anda. Terima kasih atas pengertian Anda, kami akan terus meningkatkan kualitas layanan kami.

Jika Anda berminat menjadi mitra kami, dan memiliki bandwidth yang tidak terpakai, silakan hubungi kami.
