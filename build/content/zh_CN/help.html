Blokada阻止较少的广告。

如果您怀疑Blokada阻止较之前更少的广告，请刷新您的列表。 要执行此操作, 请点击 黑名单 , 然后在均衡器图示右上角选择 刷新过滤器 并点击 继续 。 Usually it happens because new servers are added to the existing ones that are not (yet) identified and added to the blacklists, or the new list is not downloaded to your phone, therefore they are not (yet) blocked.
Alternatively you can enable more host lists or add more restrictive filters in the Blacklist menu.

Please note that Blokada uses 3rd party lists, therefore if the app itself worked earlier and blocked ads, without changes it should behave the same way.

应用程序无法按预期工作。

如果您可以看到某个应用程序不像预期的那样工作, 不加载其内容或类似的行为不端行为, 请转到主屏幕, 选择 白名单 , 在屏幕底部点击 + 符号, 选择 应用程序 并键入其名称。 最后点击 保存 。 有了这个，Blokada不会过滤这个应用程序中的网址。

Please note that Blokada 3.4 and earlier releases had a bug which caused that a freshly installed app wasn't recognized in Blokada, therefore they couldn't be set on the Whitelist. 此错误已在3.5及更高版本中修复。

我想阻止给定的网站。

您可以阻止个人网站不在主机列表中。 要执行此操作, 请转到主页上的 黑名单 , 点击屏幕底部的 + 号, 点击 主机 并键入网站的主机名, 不带 http 和 www 部分 (例如: myexample.com )。

YouTube 上的广告没有被阻止!

屏蔽YouTube上的广告非常困难, 因为谷歌使用加密以及随机生成的主机来播放广告。 Therefore, it's much easier to block ads within the YT app as with an external tool (like Blokada). 功能多样化的NewPipe和YouTube Vanced是官方YouTube应用程序的优秀替代品，请比较哪个更适合您。

我安装后无法启动Blokada。

造成这种情况的原因有两个。 首先，请检查您是否有活动的VPN连接，以及是否启用了永久在线VPN选项。 要做到这一点，请转到手机的设置，在网络部分搜索VPN，点击现有VPN旁边的设备。 将其禁用，断开与活动的VPN连接并尝试重新启动Blokada。

如果您没有其他VPN连接，请检查Blokada是否启用永久在线，如果是，请检查您是否使用wifi或移动数据连接到互联网。 如果只启用在线并且您没有互联网连接，则Blokada不会启动。

如果没有帮助，请重新启动手机，有时也会有所帮助。

Blokada在一段时间后不起作用/停止工作。

第一步，打开在Blokada中永久在线 在这种情况下，Blokada会尽量保持在线状态，但请注意，您的系统仍然会影响此行为。 如果你设置了它，你仍然会看到Blokada一次又一次地停下来，请检查你的系统设置并关闭Blokada的每个电池和内存优化。 不幸的是，它在每个系统中都有所不同，因此我们不能说在哪里做。 请检查您手机的设置。 此类行为的其他来源可能是您正在使用内存清理程序。 请在白名单或排除列表中设置Blokada。 如果没有这个步骤对您有帮助，请报告一个错误（请参阅下面的操作步骤）。

无法将 EasyList/AdBlock Plus list导入 Blokada。

如果尝试导入与您的 PC 或笔记本上的 AdBlock 加扩展非常配合的列表 (例如 EasyList), Blokada 将报告一条错误消息。 这是因为 Bloakda 使用其他方法来阻止广告。 虽然总部基地阻止和隐藏网站的元素或部分, Blokada 不能这样做, 因为它不是浏览器中的插件, 而是将访问重定向到 ad 服务器到一个不可路由的地址 (0.0.0.0), 因此它需要一个主机名 IP 列表。

Blokada是如何工作的？

Blokada是手机的网络接口(WiFi或手机) 与您使用的应用程序之间的薄层。 To be able to filter every outgoing traffic, it generates a local (no 3rd party service in use) VPN connection and simply blocks any requests towards the hosts defined in the blacklist filters. 这种阻塞是通过重定向完成的，服务于广告的网站的IP地址将被电话解析为不可路由的元地址，对于0.0.0.0，在没有web服务正在运行的情况下，因此不会显示任何内容。 有了这个，Blokada可以保存数据计划和电池寿命。 这些过滤器会自动更新，并包含已知用于投放广告，跟踪，垃圾邮件，恶意软件以及其他有害或危险内容的主机。 您可以使用您自己的自定义过滤器和白名单来配置Blokada，以确保您要使用的服务不会被阻止，并且您不会遇到任何中断。

Blokada启用时可以使用另一个VPN吗？

不幸的是，这是一个系统限制; 一次只能有一个活动的VPN连接。 如果您加入其他服务，Blokada将退出。

Blokada有哪些选择？

正如你肯定看到的那样，Blokada在主屏幕上有一些在第一次启动时没有详细说明的选项。 请让我们现在就做。 你可以看到以下内容

  • 黑名单：这是包含要阻止的服务器的主机文件列表。 请注意，使用多个列表可能会影响网络和系统速度。
  • 白名单：白名单是例外列表，您可以在此处定义不应被阻止的主机或Blokada不会过滤URL的应用程序。 Blokada默认安装了几个白名单应用程序。
  • 永久连接：此选项用于确保Blokada不会被系统暂停或停止。 请注意您的系统可以覆盖它，确保您的设置将让Blokada在后台运行。
  • 通知：启用后，您会在广告被屏蔽时收到通知。 它默认为开启，但如果您收到很多通知，请将其关闭。
  • 仅上网时：启用后，如果没有活动连接，Blokada将退出，并且每当连接回来时都会重新启动。 如果禁用，Blokada将始终在后台运行(即使没有网络可用)。 请注意，该选项可能无法在每台设备上使用。
  • 在开机时启动：启用后，Blokada将在系统启动后启动。 请注意，此选项有系统级配置，因此无法保证Blokada可以在启动时启动。 在这种情况下，请检查您手机的设置。 默认配置已启用，如果您想手动启动Blokada，请禁用此选项。
  • DNS：如果您怀疑某些网站或服务被阻止，或者您认为您的Internet提供商收集有关您的intenet使用情况的信息，请从列表中选择一台安全的DNS计算机。 默认情况下，Blokada使用与您的系统相同的DNS服务器。

不可配置的选项：

  • 最新的：如果你看到它，你正在运行最新版本的Blokada。 保持应用程序最新，因为它具有更少的错误和更多的功能！
  • #广告已拦截：Blokada第一次启动后阻止的不需要的内容数量
  • # hosts：提供广告，恶意软件和其他不需要的内容的服务器数量

如果我仍然看到一些广告，我该怎么办？

有些广告将通过Android上的任何广告拦截器进行展示。 特别是YouTube广告或某些全屏弹出式广告很难阻止。 如果您认为比以前更多的广告即将到来，请尝试以下步骤：

  • 检查顶部状态栏中是否存在VPN密钥图标。 如果没有，开启Blokada。
  • 打开Blokada并检查它是否处于活动状态并且不报告任何错误。
  • 检查主机计数器是否不等于零。 如果是，请选择一些黑名单过滤器，以便阻止广告。 如果计数器仍然为零，则报告错误。
  • 确保您的白名单中没有太多应用 - 也许这就是您看到广告的原因。
  • 尝试重置您的过滤器(转至黑名单屏幕并从顶部的“快速操作”菜单中选择恢复默认值)。
  • 如果您在浏览器中看到广告，请停用浏览器设置中的任何类型的流量压缩或优化。
  • 取消激活Blokada，将其关闭(通过从最近屏幕刷出)，然后重新打开它。
  • 如果您最近更改了设置(过滤器，白名单等)，请记住在新配置开始工作之前需要时间。 给它一些时间。
  • 清除Blokada的应用程序数据(在系统设置中)，或重新安装。
  • 重新启动您的设备。
  • 您的设备可能不受支持。 请检查设备数据库。
  • 如果这些步骤都没有帮助，请报告错误(请参阅下文)。

什么是 DNS, 我应该选择哪一个？

缩写 DNS 代表域名系统，该服务器负责将域名解析成(例如：myexample.com)成 IP 位址，IP地址则用来电脑之间的通讯。 您的互联网服务提供商拥有自己的DNS服务器；Blokada配置为使用系统默认值。 如果您怀疑您的ISP收集有关您的互联网使用情况的日志，则应该使用此列表中的计算机，但您担心某些隐私风险，或者某些网站或服务被阻止。 没有“最好的”，一切都取决于你的偏好和位置，因此如果你想使用自定义DNS服务器，请阅读他们的描述，在互联网上搜索并阅读反馈。

我无法安装更新，我该怎么办？

首先，请检查您的Android版本是否为5或更新版本。 Blokada 3.0和更新版本不支持Android 4及更早版本。 要查看版本，请前往手机的设置，找到关于手机并寻找 Android 版本

如果您的手机运行Andriod 5或更新版本，请重新启动您的设备。 这听起来很奇怪，但有时它有用。 如果没有解决问题，请卸载Blokada并重新安装。 这可能是，您的设备不受支持。 请检查设备数据库。

如果以上任何步骤都无效，请加入我们在go.blokada.org/chat上的聊天获取帮助或报告错误(请参阅下文)。

我可以隐藏 VPN 通知吗？

很不幸的是，不行。某些设备可能在VPN处于活动状态时显示令人讨厌的VPN通知(钥匙图标)，并且无法隐藏它。 如果可以，请尝试将设备更新到Android 8(奥利奥)，因为较新的Android版本没有此状态报告。

Blokada 能看到我的私人数据吗？

不会。 Blokada不会读取或篡改您的私人信息，它只会监控您的设备向哪些主机发送请求，并阻止与过滤器匹配的主机。 每当Blokada阻止任何事情时，您会看到一个通知(您可以关闭)。 Blokada无法访问您的加密流量。

Blokada是否消耗很多电量？

不会。 根据您的设置，电池的影响是最小的，在大多数新设备上，尤其是在棉花糖(Android 6.0) 或更新的设备上，您不会注意到电量的消耗。 但是，一些不幸的设备可能会增加电池消耗。 详情请看设备数据库。

Blokada 会减慢我的设备吗？

不能。 根据您的设置，Blokada可以非常高效。 事实上，您的设备的性能和电池寿命可能会明显提高，因为Blokada可以帮助您避免下载和处理您不需要或不在意的内容。 但是，使用某些较旧设备的用户可能会遇到性能下降。 详情请看设备数据库。

Blokada 真的可以免费使用吗？

是。 Blokada 开始是我的爱好，仍然主要由我开发。 这是我回馈社区的方式。 Blokada是完全开源的，它将永远保持自由。 也就是说，开发它需要很长时间，所以请考虑以任何方式支持该项目(见下文)。
