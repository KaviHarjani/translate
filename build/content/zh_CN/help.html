广告过滤

Blokada 最显著的功能就是广告屏蔽。 它是手机的网络接口(WiFi或手机) 与您使用的应用程序之间的薄层。 为了能够过滤每个字节的传出流量，它会建立一个VPN连接，然后拦截所有在黑名单中定义的主机所发出的请求。 当应用尝试访问被屏蔽的主机名时，应用会收到一个简单的“未找到主机名”报告，于是屏蔽工作完成了。 与此同时，Blokada 减少了数据使用、延长了电池寿命和使用时间。 过滤器自动更新。 您可以使用您自己的自定义过滤器和允许名单来配置Blokada，以确保您要使用的服务不会被阻止，并且您不会遇到任何中断。
此功能是免费提供的，它将保持免费。 享受使用Blokada！

如果某个应用无法按预期运行，不加载其内容或遇到类似的不当行为， 请滑动打开Blokada菜单，点击广告屏蔽， 选择 允许的Hosts， 点击顶部的+登录并添加一个单个hostname或一个包含您想要绕过过滤的所有hosts的列表文件。

You can block an individual website what isn't on a host list. 要做到这一点，请向Blokada的菜单滑动，点击广告屏蔽， 选择 被屏蔽的主机 并点击屏幕顶部的 + 签字。 在这里，您可以添加一个单个主机名或一个包含您想要屏蔽的所有主机的列表文件。

更改 DNS

Blokada 中的其他常用函数正在更改 DNS 服务器。 该应用程序提供了可供选择的预定义DNS服务器的列表。
   首字母缩写词DNS代表“域名系统”，该服务器负责将域名（例如： myexample.com ）解析为IP地址，IP地址用于在计算机之间进行通信。 您的互联网服务提供商拥有自己的DNS服务器；Blokada配置为使用系统默认值。 如果您怀疑您的ISP收集有关您的互联网使用情况的日志，则应该使用此列表中的计算机，但您担心某些隐私风险，或者某些网站或服务被阻止。 没有"最佳"，一切都取决于您的偏好和位置，因此如果您想使用自定义的DNS服务器， 请阅读他们的描述、在互联网上搜索他们并阅读反馈。
此DNS部分将与VPN 活动或不活动 所以您所选择的DNS将是使用或不使用 VPN 服务的DNS。 也考虑到 v4 Blokada 不支持DoH/Dot，但这将在v4发布后不久执行。

Blokada有什么可用设置？

正如你肯定看到的那样，Blokada在设置屏幕上有一些选项。 你可以看到以下内容

  • 通知：启用后，您会在广告被屏蔽时收到通知。 默认开启，但如果您收到太多通知，可以关掉它。
  • 开机启动: 启用后， Blokada 将会在系统启动后自动开启。 请注意，此选项有系统级配置。因此，无法保证 Blokada 可以开机启动。 在这种情况下，请检查您手机的设置。 默认配置已启用，如果您想手动启动Blokada，请禁用此选项。
  • Settings location: where do you want to store the configuration: in the internal or external storage? 如果您选择外部存储，文件可以在 /Download/Blokada 目录中找到。
  • 背景动画：您可以启用或禁用背景动画及波浪显示。 白色波浪代表被允许的主机查询，而橙色的波浪则代表被拦截的查询。
  • 请求日志记录器：启用时， Blokada 将生成一个 CSV 文件并记录被应用允许或拦截的请求。
  • 保持存活：此选项用于确保 Blokada 不会被系统暂停或停止。 请注意您的系统可以覆盖它，确保您的设置可以让 Blokada 在后台运行。
  • 看门狗：当启用时，如果没有网络连接，Blokada 将会退出，当网络重新连接时，Blokada 再次启动。 如果禁用，Blokada 将始终在后台运行(即使没有网络可用)。 请注意，该选项可能无法在每台设备上使用。
  • 主动唤醒：如果您的设备因系统的省电功能而停止或杀死 Blokada ，则启用此选项。 即使此选项能够让 Blokada 在大部分设备上保持存活，系统总是拥有更高的权限且能停止未排除内存或电池优化的应用。

安装后无法启动 Blokada。

造成这种情况通常有两个原因。 首先，请检查您是否有活动的VPN连接，以及是否启用了VPN永久在线选项。 To do it, go to your phone's Settings and search for VPN in the network section, tap on the gear next to the existing VPN. 将其禁用，断开与活动VPN的连接并尝试重新启动 Blokada 。

如果您没有其他 VPN 连接，请在 Blokada 的高级设置中检查 看门狗是否启用，如果已经启用，请检查您是否已通过无线网络或移动数据连接到互联网。 如果看门狗 已被启用而您没有网络连接，Blokada 不会启动。

如果这没有用，请重启您的手机，这有时也会有所帮助。

Blokada 屏蔽的广告比较少。

如果您怀疑 Blokada 屏蔽的广告比之前变少了，请刷新您的屏蔽列表。 要执行此操作，请点击 黑名单 ，然后在均衡器的右上角选择 刷新过滤器 并点击 继续 。 通常这是因为新服务器被添加到尚未标识列表并添加到黑名单, 或是因为新列表还未下载到您的手机上，因此这些广告还未被拦截。
或者, 您可以在黑名单菜单中启用更多 host 列表或添加更多过滤器。

请注意， Blokada 使用第三方列表。因此，如果此前应用正常工作且拦截了广告，那么在没有改变的情况下，应用的行为应当相同。

Blokada 屏蔽不了 Chrome 中的广告。

如果您正在使用 Google Chrome，您需要禁用精简模式和 Async DNS 。 谷歌默认使用这两种技术，它们干扰 Blokada 的主机屏蔽。
1. 要禁用精简模式，请点击 Google Chrome 中的三个垂直点，进入设置，搜索 ”精简模式“ 或 "data saver" 并禁用它。
2. 要禁用 Async DNS ，请在您的浏览器中打开页面"chrome://flags"，搜索DNS，并禁用 Async DNS 功能。
3. 在您的设备上，前往设置 -> 应用程序 -> Chrome -> 存储和清除缓存。
4. 重启！

YouTube中的广告未被屏蔽！

屏蔽 YouTube 上的广告非常困难, 因为谷歌使用加密以及随机生成的主机来播放广告。 因此，直接在 YouTube 应用中拦截广告比使用外部工具(如 Blokada )更容易。 功能丰富的 NewPipe 和 YouTube Vanced 是 YouTube 官方应用程序的优秀替代品，请确认哪个更适合您。

Blokada 在一段时间后不起作用/停止工作。

第一步，在 Blokada 中开启保持存活。 在这种情况下，Blokada 会尽量保持在线状态，但请注意，您的系统仍然可以影响此行为。 如果你设置好了它，并且仍然看到 Blokada 一次又一次地停了下来，请检查你的系统设置并关闭 Blokada 的所有电池和内存优化。 不幸的是，它在每个系统中都有所不同，因此我们无法具体说明在哪里设置。 请检查您手机的设置。 此类行为的其他来源可能是您正在使用内存清理程序。 请将 Blokada 加入白名单或排除列表。 如果此步骤对您没有帮助，请报告错误（请参阅下面的操作步骤）。

无法将 EasyList/AdBlock Plus list导入 Blokada。

如果您尝试导入在您的 PC 或笔记本上的 AdBlock Plus 扩展运作良好的列表 (例如 EasyList)，Blokada 将报错。 这是因为 Bloakda 使用其他方法来拦截广告。 ABP拦截和隐藏网站的元素或某个部分，而 Blokada 做不到。因为它不是浏览器中的插件，它工作在更底层，拦截主机的访问。

不可配置的东西：

  • Up to date: if you see this, you are running the most recent version of Blokada. 保持应用程序最新，因为它具有更少的错误和更多的功能！
  • # 广告被屏蔽：自首次启动以来 Blokada 阻止的不想要的内容数量
  • # 主机：服务广告、恶意软件和其他不想要的内容的服务器数量

如果我仍然看到一些广告，我该怎么办？

有些广告能绕过 Android 上的任何广告拦截器进行展示。 尤其是 YouTube 广告或一些全屏弹出广告很难拦截。 如果您认为广告比以前更多，请尝试以下步骤：

  • 检查顶部状态栏中是否出现 VPN 密钥图标。 如果没有，请启动 Blokada 。
  • 打开 Blokada 并检查它是否处于激活状态且无任何错误报告。
  • 检查主机计数器是否等于零。 如果是，请选择一些黑名单过滤器，以便阻止广告。 如果计数器仍然是零，请在系统设置 - 应用程序 - Blokada 中授予存储访问权限，然后清除数据和缓存并重启设备。
  • 请确保您在允许列表中没有太多的应用 - 也许这就是您看到广告的原因。
  • 尝试重置您的过滤器：前往”广告拦截“，点击”列表设置“，点击”过滤列表“(不在同步)，选择丢弃。
  • 如果您在浏览器中看到广告，请在浏览器设置中停用任何类型的流量压缩或优化。
  • 停用 Blokada ，关闭它（通过从系统最近任务屏幕上划出它），然后重新打开它。
  • If you recently changed your settings (filters, allowlist, etc), remember it takes some time before the new configuration starts to take effect. 给它一些时间。
  • 清除 Blokada 的应用数据（在系统设置中），或重新安装它。
  • 重新启动您的设备。
  • 如果这些步骤都没有帮助，请报告错误(请参阅下文)。

我无法安装更新，我该怎么办？

首先，请检查您的 Android 版本是否为 5 或更高版本。 Blokada 3.0 和更新版本不支持 Android 4 及更早版本。 要查看版本，请前往手机的设置，找到关于手机并寻找 Android 版本

如果您的手机运行 Android 5 或更高版本，请确保您尝试从您最初安装 Blokada 的同一来源安装更新。 不同的仓库添加它们自己的签名密钥，因此更新可能失败。

重新启动您的设备。 这听起来很奇怪，但有时却有用。 如果那不能解决问题，请卸载 Blokada 并再次安装。

如果以上任何步骤都无效，请通过go.blokada.org/chat在 Telegram 上加入我们的聊天室以获取帮助或报告错误(请参阅下文)。

我可以隐藏 VPN 通知吗？

很不幸的是，不行。某些设备可能在 VPN 处于激活状态时显示令人讨厌的 VPN 通知（一个钥匙图标），并且无法隐藏它。 如果可以，请尝试将设备更新到 Android 8（奥利奥），因为较新的 Android 版本没有此状态报告。

Blokada 能看到我的私人数据吗？

不能。 Blokada不会读取或篡改您的私人信息，它只会监控您的设备向哪些主机发送请求，并阻止与过滤器匹配的主机。 Every time Blokada blocks anything, you will see a notification (that you can turn off). Blokada无法访问您的加密流量。

Blokada是否消耗很多电量？

不会。 根据您的设置，电池的影响是最小的，在大多数新设备上，尤其是在棉花糖(Android 6.0) 或更新的设备上，您不会注意到电量的消耗。 然而一些不幸运的设备可能会增加电池消耗。

Blokada 会减慢我的设备吗？

不会。 取决于您的设置，Blokada可以非常高效。 事实上，您的设备的性能和电池寿命可能会明显提高，因为Blokada可以帮助您避免下载和处理您不需要或不在意的内容。 但是，使用某些较旧设备的用户可能会遇到性能下降。

Blokada 真的可以免费使用吗？

是！ Blokada开始作为一个爱好项目，这是回归社区的一种方式。 该应用是开源的，广告过滤功能是免费的，它将永远保持这种状态。 也就是说，开发它需要很长时间，所以请考虑以任何方式支持该项目(见下文)。
