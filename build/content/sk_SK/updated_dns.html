Blokada Slim v4.5

Vaša Blokada bola práve aktualizovaná!

Existujú dve vydania Blokady. To, ktoré používate, dostupné na Google Play, sa nazýva Blokada Slim (staré meno: DNS changer by Blokada). Toto nie je blokátor reklám. Toto vydanie umožňuje zmenu vašich DNS serverov, a taktiež používanie našej novej VPN služby (voliteľné). Open source verzia Blokady je dostupná na našej webovej stránke.

The 4.5 release is a maintenance update with improvements for stability, and user experience. For more info about the changes check our changelog.
