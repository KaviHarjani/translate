我們想要協助您的第一步驟：

  1. 目前正在使用預設的 DNS 伺服器。 輕觸預設的 DNS 圖示來修改。
  2. Blokada 被設定為當手機開啟時啟動。 你可以觸碰在開機時啟動來關閉。
  3. Blokada 只會在連上網路時運作。 你可以關閉僅上網時來變更。
  4. 在白名單中，你可以定義哪些應用程式不使用你的 DNS 伺服器。 他們將改用裝置預設的 DNS 伺服器。
  5. 點按右上角您可以找到更多選項和幫助選單，您可以在這裡獲得有關 Blokada 中不同選項的詳細資訊。
