廣告阻擋

Blokada 最著名的功能是廣告攔截。 它是個介於手機的網路介面（WiFi 或行動網路）及你使用的應用程式之間的一個薄層。 為了過濾每個連出的流量，它使用了您裝置的 VPN API，並阻擋所有定義在封鎖清單過濾規則的所有請求。 當應用程式嘗試存取被封鎖的主機名稱，此封鎖就會回應「找不到主機名稱」給應用程式。 因為如此，Blokada 可以同時節省您的行動網路流量、電池壽命與時間：因為其並未使用重新導向。 過濾器會自動更新。 您可以使用您自訂的過濾規則與白名單來設定 Blokada 以確保您想要使用的服務不會被封鎖，且不會遇到任何中斷的問題。
此功能免費提供，也將會保持免費。 盡情享受 Blokada 吧！

如果應用程式並未如預期般運作、無法載入其內容或某些類似的不當行為，請向上滑以開啟 Blokada 選單，然後輕點廣告阻擋、選擇已允許的主機，點選頂部的 + 號，並新增一個主機名稱或包含所有您想要略過過濾的主機清單檔案。

你可以阻擋主機清單上沒有的單一網站。 要這麼做，請向上滑以開啟 Blokada 的選單，在廣告阻擋上輕點，選擇已封鎖的主機並輕點在畫面頂部的 + 號。 這裡，您可以新增一個主機名稱或包含所有您想要封鎖的主機清單檔案。

變更 DNS

另一個 Blokada 的常用功能是變更 DNS 伺服器。 這個應用程式會提供一個預設的 DNS 伺服器清單來供您選擇。
DNS 是網域名稱系統 (Domain Name System) 的縮寫，而他的伺服器負責將網域名稱 (如: myexample.com) 解譯成機器之間通訊時所使用的 IP 位址。 你的網路服務供應商擁有自己的 DNS 伺服器；Blokada 設定使用系統預設值。 如果你懷疑網路服務供應商蒐集你的網路使用紀錄，擔心一些隱私上的風險，又或者是有些網站或服務被阻擋，那就使用在此清單內的主機吧。 這並沒有所謂的「最佳選擇」，一切取決於您的偏好及地點。因此在你想要使用自訂的 DNS 伺服器前，請先詳讀他們的說明，還有在網路上搜尋與其相關的反饋。
DNS 的部分不論 VPN 是否使用中都會運作，所以您選擇的 DNS 不論 VPN 服務是否啟用都會被使用。 目前 v4 Blokada 尚未支援 Doh/Dop，但會盡快在正式版推出後完成這個功能。

在 Blokada 中有哪些地方可以設定？

正如同你所看到的，Blokada 在設定有一些選項。 你可以往下看看：

  • 通知: 啟用後，您會在有廣告被阻擋時收到通知。 這個選項預設是開啟，但如果你收到太多通知，請關閉它。
  • 開機時啟動：啟用後， Blokada 會在系統開機後啟動。 這邊要注意的是這個選項還需要有相關的系統設定，因此這並不保證 Blokada 真的能夠在開機時啟動。 這種情況下，請檢查你的手機設定。 預設是啟用的，如果您希望手動啟動 Blokada，停用這這個選項。
  • 設定儲存位置：您想要將設定檔儲存於：內部或是外部儲存裝置。 如果您選擇了外部儲存裝置，這些檔案將被儲存在 /Download/Blokada 這個資料夾。
  • 背景動畫：您可以選擇啟用或是停用背景的波紋動畫。 白色波紋代表被允許通過的域名請求，而橘色的代表被阻檔的。
  • 請求記錄器: 當啟用時， Blokada 會產生一個 CSV 檔，並將所有的請求，不論允許或阻擋，都記錄在裡面。
  • 維持運行狀態：這個選項是用於確保 Blokada 不會被系統暫停或是停止。 這邊要注意的是您的系統可以覆寫該項設定，所以請確認您的設定是允許 Blokada 在背景運作的。
  • 看門狗：當啟用後，Blokada 會在斷線時關閉，在恢復連線時重新啟動。 停用時，Blokada 將會持續在背景執行（即使沒有網路）。 請注意，此選項或許無法在所有裝置上運作。
  • 主動喚醒: 如果您的裝置會因為系統的省電設定暫停或關閉 Blokada，啟用這個選項。 即便這個選項的目的是讓 Blokada 能在絕大多數裝置上維持運作，系統仍有較高的優先權，能在這個程式並未排除於記憶體或電池最佳化選項時停止程式。

我無法啟動 Blokada

有兩個常見原因會導致這樣的狀況。 首先，請先檢查您是否已經連上其他 VPN 裝置，如果有的話，是否已開啟 永久連線的 VPN 。 檢查方式是先前往手機的設定，在網路的部分搜尋 VPN。點擊 VPN 後找尋以啟動的連線。 停用它，並切斷連線中的 VPN ，再重新啟動 Blokada。

若您未使用其他 VPN 連線，請檢查 Blokada 的 看門狗 功能是否已在「進階設定」啟用。假如已經啟用，請檢查您是否連線到 Wi-Fi 或行動數據連線。 若已啟用 看門狗 且未連線到網路，則 Blokada 不會啟動。

如果沒有產生作用，有時重新啟動您的手機也是不錯的選擇。

Blokada 阻擋的廣告很少。

如果您覺得 Blokada 阻擋的廣告比以往的還少，請重新整理清單。 請點擊黑名單，在右上角的等化器圖示中，選擇重新整理過濾器並輕點繼續。 通常會發生這種情況，可能是因為有新的伺服器加入到既有的伺服器之中，且還沒被辨識出來並加入到黑名單中，或是新的清單還沒下載到您的手機上，因此還沒被阻擋。
您可以在黑名單選單中啟用更多主機清單或是加入更多限制性的過濾器。

請注意 Blokada 使用第三方的清單，因此若應用程式在稍早的時候可以正常運作並阻擋廣告，在沒有變更的狀況下它應該會有相同的行為。

Blokada 不會阻擋 Chrome 的廣告。

如果您正在使用 Google Chrome 您需要停用 Data Saver 及 Async DNS。 Google 預設啟用這兩個技術，而這些會妨礙 Blokada 的主機阻擋功能。
1. 如果要關閉 Data Saver，輕點 Google Chrome 裡面的垂直三點，進到設定，尋找 「節省數據」或是「精簡模式」並關閉它。
2. 如要關閉 Async DNS，在瀏覽器開啟 "chrome://flags "，搜尋 DNS 並停用 Asynchronous DNS 這個功能。
3. 請進入您裝置的《設定》->〈應用程式〉-> Chrome -> 〈儲存空間〉並清除快取。
4。 重新啟動!

無法阻擋 YouTube 中的廣告！

在 YouTube 上阻擋廣告是非常困難的，因為 Google 使用了加密且隨機生成的主機來播送廣告。 因此，使用外部工具（如 Blokada）在其他應用程式上會比較容易阻擋廣告。 NewPipe 或 YouTube Vanced 都是多功能的 YouTube 官方替代品，請看看哪個軟體對您比較合用。

Blokada 過一段時間就會停止運作。

第一步，打開在 Blokada 中的持續連線。 在這個情況下，Blokada 將會試著保持連線，但請注意，系統仍然可以會影響到這個行為。 如果已經設定此項，但 Blokada 仍會在一段時間後停止運作，那請檢查系統設定，並關閉每個作用於 Blokada 的電池及記憶體最佳化設定。 非常不巧的，在每個系統都不一樣，所以我們無法講要在哪裡做。 請檢查您手機的設定。 造成這種行為的其他原因，可能是因為使用記憶體清理工具。 請把 Blokada 設定在白名單中或排除清單中。 如果這些步驟無法協助您的情況，請回報錯誤(看下方如何去做)。

我無法匯入 EasyList/AdBlock Plus 清單到 Blokada 中。

若您試著匯入在您的電腦上運作良好的 AdBlock Plus 附加元件的清單（例如 EasyList），Blokada 會回報錯誤。 這是因為 Blokada 使用了其他方法來阻擋廣告。 Blokada 並不能如 ABP 那般，阻擋並隱藏在網頁上的元素或區塊。因為它並非瀏覽器的附加元件，是採用較低層級的運作方式來阻擋主機。

無法設定的事物:

  • 更新至最新：如果您看見它，則表示正在運作最新版本的 Blokada。 請將應用程式維持在最新狀態，因為會有比較少的bug跟更多的功能！
  • 已阻擋# 廣告: 首次啟動Blokada時會顯示不必要內容阻擋次數
  • # 主機：提供過濾廣告、惡意軟體及其他可能有害內容的伺服器數量。

如果我仍然看到廣告，該怎麼辦？

有些廣告會繞過 Android 上的廣告阻擋器。 尤其是難以阻擋的 Youtube 廣告或全螢幕彈出視窗。 如果你認為出現的廣告比以前還多，請嘗試以下步驟：

  • 檢查 VPN 鑰匙圖案是否在上方的狀態列顯示。 如果沒有，開啟 Blokada。
  • 點開 Blokada 並檢查是否已經啟動並且沒有回報任何錯誤。
  • 檢查主機計數器是否等於零。 如果是，選擇黑名單過濾器，以便阻擋廣告。 若計數器仍為 0，請在「系統設定」→「應用程式」→「Blokada」的應用程式權限中授予「儲存空間存取權限」，在清除資料與快取後重新啟動。
  • 確保您的白名單內沒有太多應用程式 - 也許這就是您看到廣告的原因。
  • 嘗試更新過濾器：前往「廣告封鎖」，點擊「列表設定」，點擊「過濾器列表」（不在同步中），選擇「丟失」。
  • 如果您仍然在瀏覽器中看到廣告，請進入瀏覽器的設定，停用任何類型的節省數據壓縮或最佳化。
  • 取消 Blokada，將其關閉 （ 由最近啟動畫面上刪除），再重新啟動。
  • 如果你最近更改設定（過濾器、白名單等）。注意，要一段時間才會產生作用。 請稍待片刻。
  • 清除 Blokada 的應用程式資料 （在系統設定裡） 或重新安裝。
  • 重開機。
  • 如果這些步驟都沒有幫助，請回報錯誤(見下文)

我無法安裝更新，我該怎麼辦？

首先，請檢查你的 Android 版本是否為 5 或以上的版本。 Blokada 3.0 及新版不支援 Android 4 及以下的版本。 要檢查版本，先前往手機的設定，拖曳到關於手機並搜尋 Android 版本

如果您的手機是使用 Android 5 或是以上的版本，請確認您是從第一次安裝 Blokada 時的同樣來源取得及更新。 不同的軟體庫會簽署自己的金鑰，而造成更新失敗。

重新啟動裝置。 這聽起來很奇怪，但有時候會很有幫助。 如果無法解決問題的話，請解除安裝 Blokada 並重新安裝。

如果這些步驟都沒有幫助，請加入我們的 Telegram 討論 go.blokada.org/chat 以獲得幫助。或著您可以回報錯誤（見下文）。

我能隱藏 VPN 通知嗎？

非常遺憾的，答案是不行。當啟動 VPN 時，有些裝置會顯示惱人的 VPN 通知（鑰匙狀圖示），而此圖示無法隱藏。 如果可以，請嘗試將您的設備更新至Android 8 (Oreo)，因為較新的Android版本沒有此狀態回報

Blokada 能看到我的隱私資料嗎？

不會。 Blokada 不會讀取或竄改你的隱私資訊，只有監控裝置正在向哪些主機發出請求，並阻擋與過濾器符合的主機。 每次 Blokada 阻擋任何請求，您會收到通知（您可以關閉此功能）。 Blokada 不會讀取你加密的網路流量。

Blokada 會不會使用很多電力？

不會。 您的設定應該對電池影響不大。而且在大部分裝置，尤其是在 Marshmallow (Android 6.0) 以上的版本，您不會注意到電池額外的耗損。 然而，有些裝置仍然會不幸的遇到電池消耗增加的情況。

Blokada 會讓我的裝置變慢嗎？

不會的。 取決於您的設定，Blokada 可以用很高效率的方式運作。 事實上，因為 Blokada 幫您免於下載及處理不需要或不在意的內容，這對裝置性能與電池壽命，可能反而有明顯的改善。 然而，某些較為舊型的裝置可能會遇到效能降低的狀況。

Blokada 真的是免費使用嗎？

是的！ Blokada 是因為興趣而建立的專案，他是一種回饋社群的方式。 這個軟體是開源的，其中主要的廣告過濾功能是免費的，並永遠如此。 也就是說，開發這個軟體會耗費許多心力。請試著使用您方便的方式來幫助這個專案(來看看下面吧)。
