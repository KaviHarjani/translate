స్వాగతం మరియు Blokada ఉపయోగించినందుకు ధన్యవాదాలు!

Blokada ఉచితము మరియు ఓపెన్ సోర్స్ మరియు ఇది మీ గోప్యతను గౌరవిస్తుంది. దీన్ని ఉపయోగించడం ద్వారా మీరు సేవా నిబంధనలు మరియు గోప్యతా విధానాన్ని అంగీకరిస్తారు, మీరు ఎల్లప్పుడూ ఇక్కడ కనుగొంటారు .

మేము మీకు మొదటి దశలో సహాయం చేయాలనుకుంటున్నాము:

  1. By default you'll be notified when an ad is blocked. You can change it by setting Notifications to off.
  2. Blokada is set to start when your phone is started up. You can turn if off by tapping on Start on boot.
  3. Blokada will be running only when you are connected to the internet. You can change it by setting Online only to off.
  4. In the Blacklist you can find the lists that contain the ads-servers to block.
  5. In the Whitelist you can define which apps or websites shouldn't be checked and blocked.
  6. Tapping in the top right corner you can find more options and the Help menu where you can get detailed information about the different options in Blokada.
